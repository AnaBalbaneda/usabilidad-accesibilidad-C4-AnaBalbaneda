Declaración de Accesibilidad

La Enciclopedia Médica Ilustrada se ha comprometido a hacer accesible su sitio web, de conformidad con el Real Decreto 1112/2018, de 7 de septiembre, sobre accesibilidad de los sitios web y aplicaciones para dispositivos móviles del sector público, así como con la directiva europea y las pautas WCAG 2.1.

Situación de cumplimiento

Este sitio web es plenamente conforme con el RD 1112/2018 y las pautas de accesibilidad WCAG 2.1 nivel AA, gracias a las auditorías automáticas y manuales aplicadas durante su desarrollo para eliminar cualquier barrera de acceso.

Preparación de la presente declaración de accesibilidad

La presente declaración fue preparada en mayo de 2026.

El método empleado para preparar la declaración ha sido una evaluación manual experta llevada a cabo por la propia desarrolladora, empleando herramientas de inspección semántica como ANDI, revisión de jerarquías con HeadingsMap, simuladores de visión cromática y comprobadores matemáticos de contraste.

Observaciones y datos de contacto

Puede realizar comunicaciones sobre requisitos de accesibilidad (artículo 10.2.a del RD 1112/2018) como, por ejemplo:

  • Informar sobre cualquier posible incumplimiento por parte de este sitio web.
  • Transmitir otras dificultades de acceso al contenido.

Puede comunicar estos aspectos a través de nuestro formulario de contacto o comunicándose con la administradora del sitio: Ana Balbaneda Palau.

Medidas técnicas de accesibilidad adicionales adoptadas

Además de la normativa exigida, este sitio web incorpora las siguientes mejoras pensadas para los usuarios:

  • Identificación del idioma: Se ha definido el idioma principal del sitio mediante el atributo lang en la etiqueta html.
  • Estructura semántica: Se han empleado etiquetas de HTML5 (como header, nav, main, footer) para organizar el contenido de forma lógica.
  • Navegación facilitada: Se han incluido migas de pan y un mapa del sitio para ayudar a los usuarios a ubicarse y navegar fácilmente.

Estilos alternativos y cómo activarlos

Este sitio web ofrece múltiples hojas de estilo pensadas para diferentes necesidades visuales. Puede activarlas desde su navegador web (por ejemplo, en Firefox, yendo al menú superior Ver > Estilo de página). Las opciones disponibles son:

  • Modo Noche: Colores oscuros para reducir la fatiga visual.
  • Alto Contraste: Colores extremos para personas con baja visión severa.
  • Letra Grande: Aumento general del tamaño del texto.
  • Alto Contraste y Letra Grande: Una combinación de las dos anteriores.
  • Versión para imprimir: Preparada para ahorrar tinta y mostrar una tipografía óptima para papel al mandar el documento a impresión.

Accesibilidad en documentos descargables

Los documentos electrónicos (PDF) ofrecidos en este sitio cumplen con las pautas de accesibilidad: conservan las etiquetas de encabezados, incluyen textos alternativos en las imágenes, marcan explícitamente las cabeceras de tabla y contienen metadatos descriptivos.

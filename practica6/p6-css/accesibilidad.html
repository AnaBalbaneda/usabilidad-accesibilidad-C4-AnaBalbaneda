Declaración de Accesibilidad

Este sitio web ha sido diseñado con el objetivo de ser accesible para el mayor número de personas posible, independientemente de sus capacidades técnicas o físicas.

Medidas de accesibilidad adoptadas

  • Identificación del idioma: Se ha definido el idioma principal del sitio mediante el atributo lang en la etiqueta html[cite: 35, 299].
  • Estructura semántica: Se han empleado etiquetas de HTML5 (como header, nav, main, footer) para organizar el contenido de forma lógica[cite: 36, 223].
  • Jerarquía de encabezados: Se utilizan etiquetas de encabezado (h1, h2, etc.) de manera ordenada y sin saltar niveles para facilitar la lectura[cite: 37, 124].
  • Texto alternativo: Todas las imágenes cuentan con el atributo alt para describir su función o contenido a los lectores de pantalla[cite: 38, 318].
  • Navegación facilitada: Se han incluido migas de pan y un mapa del sitio para ayudar a los usuarios a ubicarse y navegar fácilmente[cite: 50, 509].
  • Formularios accesibles: Todos los campos de formulario están correctamente etiquetados con label y agrupados con fieldset y legend[cite: 39, 273].

Nuevas características de accesibilidad visual

  • Uso correcto de los colores: Se han elegido paletas de colores que garantizan un contraste adecuado entre el texto y el fondo para facilitar la lectura.
  • Resaltado del foco: Todos los elementos interactivos (enlaces, campos de formulario) tienen un resaltado muy evidente (cambio de color de fondo y borde) cuando reciben el foco del teclado, ayudando a la navegación sin ratón.
  • Estilos alternativos y cómo activarlos: Este sitio web ofrece múltiples hojas de estilo pensadas para diferentes necesidades visuales. Puede activarlas desde su navegador web (por ejemplo, en Firefox, yendo al menú superior Ver > Estilo de página). Las opciones disponibles son:
    • Modo Noche: Colores oscuros para reducir la fatiga visual en entornos de poca luz.
    • Alto Contraste: Colores extremos (blanco y amarillo sobre negro) para personas con baja visión severa.
    • Letra Grande: Aumento general del tamaño del texto de la web al 150%.
    • Alto Contraste y Letra Grande: Una combinación de las dos anteriores.
  • Versión para imprimir: El sitio está preparado para que, al enviarse a la impresora, se eliminen automáticamente los menús y colores de fondo, ahorrando tinta y mostrando una tipografía óptima para la lectura en papel.
